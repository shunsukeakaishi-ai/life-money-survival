# 人生資産サバイバル（MVP / Phase 2.4）

## Phase 2.4で調整した内容
- 投資連続カウント（investmentStreak）追加
- 投資連打時の生活面リスクを追加
- 投資信託/個別株/仮想通貨のイベント個性を強化
- 「売却する」を正式な月次行動に変更（売却時も1ヶ月進行・イベント発生）
- 仮想通貨の暴落ショック表示を「追加影響」に整理
- 標準クリア目標を200万円へ引き上げ、借金条件・生活見直し仕様は維持

## 投資先ごとの個性
- 投資信託: 安定寄り（積立習慣、余裕資金の窮屈さ）
- 個別株: 会社イベント・優待・配当・悪材料
- 仮想通貨: 乱高下ストレスと売り時判断

## 維持仕様
- 目標純資産200万円、借金20万円未満でクリア
- 生活見直しLv0〜3、生活費170k→160k→155k→150k
- 生活見直しクールタイム6ヶ月、リフレッシュ4ヶ月

## テスト観点
1. 投資連続が投資行動で増え、非投資行動でリセットされるか
2. 投資信託は安定しつつ連打で節約疲れ系が出るか
3. 個別株で会社イベント（決算/不祥事/配当/優待）が出るか
4. 仮想通貨で乱高下ストレス表示が自然か
5. 借金・クリア条件・生活見直し仕様が壊れていないか
6. 売却時に月次サマリー/イベントカード/イベントログが売却ターン内容に更新されるか

## Phase 2.7のバランス意図
- 150万円目標では、生活見直し・副業・投資の組み合わせでやや達成しやすくなったため、標準目標を200万円へ引き上げ
- 普通に働く＋休養＋生活見直しだけでは届きにくく、行動の組み合わせが必要な設計へ
- 副業・投資・売却・休養・生活見直しを状況に応じて回す判断を重視
- デバフを強めすぎる理不尽調整は避け、まずは目標金額で難易度を調整
- 中期イベントやエンディング分岐は将来候補として継続検討
- クリア後/ゲームオーバー後の資産推移グラフ（現金・投資評価額・借金・純資産）は将来候補
- 累計収支グラフ（本業/副業/生活費/イベント収支/投資損益）と折りたたみ家計簿ログは将来候補
- ストレス90以上は危険域として扱い、ストレス100継続でゲームオーバーに到達するMVP調整を追加
- 高ストレス時の体力減少は通常イベントとは別の「追加影響」として扱う
- 本業収入減・健康リスク本格実装・ストレス総到達回数カウントは将来候補として見送り
