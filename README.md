# 人生資産サバイバル

「人生資産サバイバル」は、**36ヶ月**のあいだ生活を守りながら**純資産200万円**を目指す、生活・仕事・副業・投資のリソース管理ゲームです。

## ゲーム概要
- 毎月1つ行動を選び、現金・投資評価額・借金・体力・ストレスを管理します。
- 収入を伸ばす行動ほど、体力やストレスへの負担が増える場合があります。
- 終了時にリザルト（結果、推移グラフ、バッジ、メーター）で振り返れます。

## 現在のクリア条件（現行仕様）
- **36ヶ月終了時点で判定**
- **純資産200万円以上**
- **借金20万円未満**
- 途中で200万円に到達しても即クリアではありません。
- 途中到達月（`firstReachMonth`）と最高純資産（`maxNetWorth`）は記録されます。

## 基本行動
- 本業に集中する
- 副業を頑張る
- 休養する
- 投資を増やす
- 借金する
- リフレッシュする
- 生活を見直す
- 売却する

## 主要システム
- 36ヶ月制
- 純資産 / 現金 / 投資評価額 / 借金
- 体力・ストレス
- ストレス危険カウント
- 副業疲労
- 本業評価
- 生活見直しLv
- 投資タイプ（投資信託 / 個別株 / 仮想通貨）
- 投資タイプ上書き禁止（売却後のみ変更可）
- 投資連続リスク
- 副業＋投資の両立疲れ
- 高ストレス疲労

## リザルト機能（実装済み）
- 今回の結果
- 最終純資産
- 目標との差
- 初回到達月
- 最高純資産
- プレイ傾向メーター
- 今回のプレイバッジ
- 詳細折りたたみ
- 資産推移グラフ（純資産）

## 遊び方説明（ゲーム内）
- タイトルカード内に折りたたみ式「遊び方」を実装しています。
- 初見向けの最小説明を表示し、細かい攻略情報はあえて載せていません。

## デバッグモード（開発用）
- 通常URLでは非表示です。
- `?debug=1` のときだけデバッグパネルを表示します。
- 主な機能:
  - state確認・編集
  - プリセット適用
  - history確認
  - サンプルhistory生成
  - リザルトプレビュー
- 開発・検証用機能であり、通常プレイ向けではありません。
