## 手動テスト観点
1. リフレッシュで「思い切ってリフレッシュ」が正しく表示されるか
2. 上部ステータスに現在月/現金/純資産/体力/ストレス/借金/生活費/Lv/リフレッシュ可否が出るか
3. 生活を見直すでLv0→1→2→3、生活費170k→160k→155k→150kになるか
4. 借金0で返済プレッシャーが出ないか
5. 借金20万円以上で投資不可、30万円以上で借入不可が効くか
