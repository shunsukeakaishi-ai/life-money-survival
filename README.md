# 人生資産サバイバル（MVP / Phase 2）

## ゲーム概要
22歳の新人会社員として36ヶ月を生き抜き、純資産100万円以上を目指す1画面ブラウザゲームです。毎月、行動を1つ選び、生活費・健康・ストレス・投資・借金を管理します。

## Phase 2で改善したこと
- UIを再構成し、画面上部に「最新イベントカード」と「行動選択」を配置
- イベントを増量し、毎月の出来事が印象に残る表示へ改善
- バランス調整（序盤の余裕削減・副業連打ペナルティ・投資条件追加）
- クリア/ゲームオーバー演出、危険状態の警告表示、資産メーターを追加

## 作成ファイル
- `index.html` : UI本体
- `style.css` : レイアウト・配色・演出
- `script.js` : ゲームロジック
- `MVP_SPEC.md` : MVP仕様と制限
- `README.md` : この説明

## ローカルで確認する方法（初心者向け）
### 最短（ダブルクリック）
1. このリポジトリをPCにダウンロード
2. `index.html` をダブルクリック
3. ブラウザでゲームが起動

### うまくいかない時（簡易サーバー）
1. Windowsで「PowerShell」を開く
2. このフォルダに移動
3. 次を実行：
   ```powershell
   py -m http.server 8000
   ```
4. ブラウザで `http://localhost:8000` を開く

## GitHub Pagesで公開する方法
1. GitHubのリポジトリを開く
2. `Settings` タブを押す
3. 左メニュー `Pages` を押す
4. `Build and deployment` の `Source` を `Deploy from a branch` にする
5. `Branch` を `main`（または公開したいブランチ）/`/(root)` にして `Save`
6. 数分待つと公開URLが表示される
