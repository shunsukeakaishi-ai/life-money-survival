# 人生資産サバイバル（MVP / Phase 2.3）

## Phase 2.3で調整した内容
- 目標純資産を100万円→150万円へ変更
- 生活見直しに6ヶ月クールタイム追加
- 生活見直しの収入を170,000→150,000へ調整
- 現金GOOD/BADイベントの頻度を少し抑制
- 体力・ストレス中心の日常イベントを追加
- UIに「目標150万円」「生活見直しクールタイム」表示を反映

## 主な仕様
- クリア条件: 純資産150万円以上 かつ 借金20万円未満
- 生活見直しLv: 0〜3
- 生活費: 170,000→160,000→155,000→150,000
- 生活見直しクールタイム: 6ヶ月
- リフレッシュクールタイム: 4ヶ月

## 手動テスト観点
1. 目標表示が150万円になっているか（タイトル/資産メーター/クリア文言）
2. 生活見直しの収入が150,000で、6ヶ月クールタイムが発生するか
3. Lv0→1→2→3で生活費が170k→160k→155k→150kになるか
4. 現金イベントが連続しすぎず、日常イベント（体力/ストレス）が増えたか
5. 借金20万円投資不可・30万円借入不可・借金20万円未満クリア条件が維持されるか
