# 人生資産サバイバル（MVP / Phase 2.1）

## 概要
36ヶ月を生き抜き、純資産100万円以上（かつ借金20万円未満）を目指す1画面ブラウザゲームです。

## Phase 2.1で改善した内容
- イベント抽選の理不尽さを軽減（行動別候補 + 状態連動 + 同イベント連打抑制）
- ストレス調整（休養/リフレッシュで立て直しやすく）
- 新アクション「思い切ってリフレッシュ」を追加
- 借金投資対策（借金20万円以上で投資不可、30万円以上で追加借入不可）
- 上部に簡易ステータスバーとリフレッシュクールタイム表示を追加

## ローカル確認方法
1. `index.html` をダブルクリックして開く
2. うまく表示されない場合は PowerShell で以下を実行
   ```powershell
   py -m http.server 8000
   ```
3. ブラウザで `http://localhost:8000` を開く

## GitHub Pages公開方法
1. リポジトリの `Settings` → `Pages`
2. Source を `Deploy from a branch`
3. `main` / `root` を選択し `Save`

## 手動テスト手順（簡易）
1. 休養を選んだ月に「残業続き」「小さなミス」が出ないことを確認
2. 借金0で「返済プレッシャー」が出ないことを確認
3. 投資残高0で投資上げ/下げイベントが出ないことを確認
4. ストレス高騰時に「思い切ってリフレッシュ」で立て直せることを確認
5. 借金20万円以上で投資不可、30万円以上で借入不可を確認

## 次にテストプレイで見るべきポイント
- 通常状態でBADイベントが偏りすぎないか
- 副業連打時にリスクが増えるか
- 投資信託は堅実、個別株は中リスク、仮想通貨は高リスクになっているか
- 借金を緊急手段として使えるが、借金投資で勝ち逃げしにくいか
