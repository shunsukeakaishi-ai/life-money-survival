# 人生資産サバイバル MVP仕様（Phase 2.3）

## 目的
- 普通プレイだけで簡単に勝てる状態を抑え、150万円目標に再調整
- 生活見直しの連打を防ぐ（6ヶ月クールタイム）
- お金イベントの偏りを抑え、体力・ストレス中心の生活イベントを増やす

## 主要変更
- クリア条件: 純資産150万円以上 かつ 借金20万円未満
- 資産メーター基準を150万円へ変更
- 生活見直し
  - 収入 +150,000（旧170,000）
  - 体力 -5 / ストレス +3 / Lv+1 / 副業連続リセット
  - 6ヶ月クールタイム追加
  - Lv3到達後は使用不可
- 生活費レベルは据え置き
  - Lv0:170,000 / Lv1:160,000 / Lv2:155,000 / Lv3:150,000

## イベント調整
- 現金が直接増えるGOODイベントの重みを下げる
- 現金が直接減る大型出費イベントの重みも少し下げる
- 体力・ストレスが変動する日常イベントを追加
  - 公園の猫と遊んだ
  - 湯船にゆっくり浸かった
  - 夜ふかしした
  - スマホを見すぎた
  - 友人と話した

## 既存維持
- 借金20万円以上で新規投資不可
- 借金30万円以上で追加借入不可

## 今回実装しないもの
- 健康リスク・入院システム（次Phase候補として保留）
