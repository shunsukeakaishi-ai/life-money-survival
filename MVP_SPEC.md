# 人生資産サバイバル MVP仕様（Phase 2.4）

## 調整意図
- 投資連打の単調さを改善
- 投資信託 / 個別株 / 仮想通貨の個性を強化
- Phase 2.7で標準目標を200万円へ引き上げ、借金条件など既存基盤は維持

## 追加仕様
- investmentStreak（投資連続カウント）を追加
  - 投資行動で+1
  - 投資以外で0にリセット
- ステータス表示に投資連続を追加
- 「売却する」を正式な月次行動として扱う
  - 売却時も1ヶ月進行、生活費/借金処理、イベント発生、勝敗判定を通常フローで実行
  - 売却後は `investmentBalance=0`、`investmentType=null`、`investmentStreak=0`、`lastInvestMonth=null`

## 投資先ごとの個性
- 投資信託: 安定系イベント（積立が習慣、余裕資金が少ない）
- 個別株: 会社系イベント（決算好調、不祥事、配当、株主優待）
- 仮想通貨: 乱高下系イベント（乱高下ストレス、利確の誘惑）

## 表示整理
- 仮想通貨の大幅下落時は、イベント名と分離して
  `追加影響: 乱高下ストレス +16` を表示

## 維持仕様
- 目標純資産200万円
- 借金20万円未満でクリア
- 生活見直しLvと生活費段階
- 生活見直し6ヶ月クールタイム
- 借金20万円以上で投資不可
- 借金30万円以上で追加借入不可

## Phase 2.7 バランス調整メモ
- 150万円目標では、副業・投資・生活見直しの組み合わせで達成しやすくなったため、標準目標を200万円へ引き上げ
- 普通に働く＋休養＋生活見直しのみでは届きにくく、複数行動の組み合わせで到達を目指す設計
- デバフ強化より先に目標金額の調整を優先し、理不尽感を抑える
- 中期イベント（繁忙期/市場不安定期/物入り期）やエンディング分岐は将来候補

## 今回未実装
- 市場ムード表示（次Phase候補）
- 健康リスク・入院システム
