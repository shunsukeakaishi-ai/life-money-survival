# 人生資産サバイバル MVP仕様（Phase 2.1）

## 目的
Phase 2で強くなりすぎた理不尽さ（BAD過多・ストレス張り付き）を緩和し、
「立て直し可能で戦略が気持ちよく回る」調整を行う。

## Phase 2.1の主な変更
- 上部に簡易ステータスバーを追加（現金/純資産/体力/ストレス/借金を即確認）
- 新行動「思い切ってリフレッシュ」を追加（強い回復、クールタイムあり）
- 行動別イベント候補を導入し、違和感ある組み合わせを抑制
- 借金なしでは借金系イベントを出さない
- 投資残高なしでは投資残高変動イベントを出さない
- 直近3件イベントの簡易クールダウン（同イベント連打抑制）

## バランス調整
- 初期現金 100,000円
- 生活費 170,000円
- 投資条件: 現金250,000円以上 + 投資後も生活費確保
- 借金20万円以上: 新規投資不可
- 借金30万円以上: 追加借入不可
- クリア条件: 純資産100万円以上 かつ 借金20万円未満
- 副業連続で負荷増 + 副業系BADイベントが出やすい
- 休養時はストレス回復を強め、回復系イベントが出やすい
- 仮想通貨は高ボラ維持、暴落ショック時にストレス増

## 新行動: 思い切ってリフレッシュ
- 効果: 収入+150,000 / 支出-80,000 / 体力+15 / ストレス-70
- 副業連続回数リセット
- クールタイム: 4ヶ月
- 現金80,000円以上で実行可

## 既知の制限
- セーブ機能なし
- 乱数シード固定なし
- 投資の同時複数保有なし（MVP簡略化）
