# 人生資産サバイバル MVP仕様（Phase 2.2）

## 目的
- Phase 2.1のバグ修正（リフレッシュ/上部ステータス空欄/undefined表示）
- 理不尽さ軽減と、生活安定ルート追加
- ストレスを「放置すると不利、対策で回復可能」な数値に調整

## 主な変更
- 新行動「生活を見直す」を追加
- 生活見直しレベル（0〜3）を追加
- 生活費をレベル連動に変更
  - Lv0:170,000 / Lv1:160,000 / Lv2:155,000 / Lv3:150,000
- 上部の判断用ステータスに、生活費・生活見直しLv・リフレッシュ可否を表示

## リフレッシュ修正
- 行動名を正式登録し、サマリーに日本語表示
- 効果: 収入+150,000 / 支出-80,000 / 体力+15 / ストレス-70
- クールタイム4ヶ月、現金80,000円未満では実行不可
- クールタイム中はボタン無効化

## 借金・投資ルール
- 借金20万円以上: 新規投資不可
- 借金30万円以上: 追加借入不可
- クリア条件: 純資産100万円以上 かつ 借金20万円未満

## イベント抽選（Phase 2.2）
- 行動別イベント候補を採用
- 借金0なら返済系イベント除外
- 投資残高0なら投資残高変動イベント除外
- 休養時は回復系・生活系を中心
- 副業連続時は副業トラブル系が増え、仕事評価系は出にくい
- 直近3イベントの重みを下げて連続発生を抑制

## 既知の制限
- セーブ機能なし
- 乱数シード固定なし
- 投資の同時複数保有なし
